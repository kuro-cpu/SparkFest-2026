KEY FEATURES AND GOOGLE TECHNOLOGY USED
• Emergency Triage- SULO can rapidly assess safety situations to determine the appropriate level of urgency, helping users prioritize their actions during a crisis.
• Resource Access- The application provides immediate, easy-to-find links and information for campus safety contacts and essential emergency resources.
• Adaptive Incident Reporting (Gemini API) – Guides students through context-aware conversations that collect complete and meaningful incident details.  
• AI Case Prioritization (Gemini API) – Generates structured summaries and assigns urgency levels to support faster human response.  

GETTING STARTED WITH SULO
SULO is designed for speed and clarity during emergencies. Follow this quick tour to see how the assistant works:
Launch the Assistant: Open SULO on your device to access the main dashboard.
Describe the Incident: Simply type your situation into the chat box. For example, "I am near the gymnasium and there is a suspicious individual following me."
Harness the Power of Gemini: Once you hit send, Gemini analyzes your input in real-time, providing an immediate risk assessment and step-by-step safety instructions.
Review the Triage: SULO displays the output generated by Gemini, including a structured summary and an urgency level, allowing you to act quickly.
Contact Support: Use the emergency quick-links provided in the interface to connect with campus security immediately.

HOW TO USE SULO
SULO is designed for speed. Follow these steps to experience the safety assistant:
Initiate the Incident: Click the main chat interface or "Report" button to open the AI assistant.
Describe the Situation: Type a brief, natural language description of what is happening (e.g., "I'm in the library and there's a small fire near the exit").
Let SULO Triage: The AI will immediately analyze your message, provide immediate safety steps, and generate a structured summary with an urgency level.
Access Resources: Use the sidebar or quick-links to contact campus security or emergency services based on the triage report provided by SULO.

SETTING UP GEMINI API KEY
To enable the AI chat features in SULO, you need to provide your own Gemini API key:

Create a file named config.js in the project's root folder.

Add your API key to the file by pasting the following line and replacing YOUR_API_KEY_HERE with your actual key:

JavaScript:
const GEMINI_API_KEY = "YOUR_API_KEY_HERE";
Save the file. The application will automatically detect this file and connect to the Gemini API.
